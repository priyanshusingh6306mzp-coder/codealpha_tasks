# CodeAlpha Machine Learning Internship

Welcome to my repository for the **CodeAlpha Machine Learning Internship**. This repository contains the machine learning projects completed as part of the internship.

## Completed Tasks

### Task 1: Disease Prediction from Medical Data
**Objective:** Predict the likelihood of heart disease using patient medical data.

**Highlights:**
- Performed Exploratory Data Analysis (EDA)
- Data preprocessing and feature engineering
- Trained multiple machine learning models
- Compared models using Accuracy, Precision, Recall, F1 Score, and ROC-AUC
- Selected the best-performing model

---

### Task 2: Credit Scoring Model
**Objective:** Predict an individual's creditworthiness using historical financial data.

**Highlights:**
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Trained and compared multiple classification models
- Evaluated models using Accuracy, Precision, Recall, F1 Score, and ROC-AUC
- Saved the best-performing model
