# Disease Prediction from Medical Data

## Objective
Predict the likelihood of heart disease using patient medical data.

## Dataset
Heart Disease Dataset

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Best Model
Logistic Regression achieved the best performance on the test dataset.
